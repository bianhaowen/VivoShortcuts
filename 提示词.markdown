# vivo快捷指令开发提示词

## 开发环境

我需要你帮我开发一个可以在vivo手机上运行的软件，请根据安卓官方推荐，先部署完整的开发环境。


## 程序开发

### 1.0

请帮我开发一个软件
功能：点击桌面图标，弹出悬浮窗，悬浮窗中包含多个按钮，点击不同按钮执行不同功能，点击悬浮窗外区域关闭软件。
按钮功能：将铃声、通知和闹钟的音量调到10%，以10为差值递增，至100%，共计十个按钮，点击按钮时，若当前模式为震动或静音，则将模式改为响铃。
UI：悬浮窗背景为圆角矩形，呈现毛玻璃效果，每个按钮为圆角正方形，颜色为渐变色，每排自左向右平滑渐变，一排五个，每个音量按钮的图标为一个喇叭加上数字，数字使用无衬线字体，整体表现出圆润、温和质感。

### 1.1

进行以下修改：
1、生成apk包名称为XCShortcuts.apk，软件的中文名称为“鑫澄的快捷指令”，英文名称为“XCShortcuts”；
2、编译打包时在release模式下进行，开启编译器优化；
3、程序当前功能不涉及网络下载、读写用户配置文件等需要写文件或缓存的功能，不要出现安装后在文件系统中乱拉屎的现象；
4、程序中调用的API尽可能使用官方推荐的、新的第一方API，尽量不使用老旧的或第三方API。

### 1.2

进行以下修改：
1、软件图标为两行英文字符，第一行为“XC”，第二行为“shortcuts”，字体使用无衬线字体；
2、点击桌面图标后非操作区域不要黑屏，效果类似于系统的通知，不影响其它区域显示当前内容。

  ，无需手动关闭）

### 2.0

进行以下修改：
1、不再使用悬浮窗模式，改为正常程序；
2、UI：背景为白灰色（偏白），按钮为圆角矩形，每个按钮微凸，按下后凹陷，每行两个按钮，共五行，自左上至右下所有按钮颜色平缓渐变，每个按钮上一个喇叭图案，喇叭图案右边是音量百分比，按钮下无需文本；
3、程序打开默认使用小窗模式，退出至后台后自动销毁。
