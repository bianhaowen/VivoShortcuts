# vivo快捷指令开发提示词

## 开发环境

我需要你帮我开发一个可以在vivo手机上运行的软件，请根据安卓官方推荐，先部署完整的开发环境。


## 程序开发

### 版本1

请帮我开发一个软件
功能：点击桌面图标，弹出悬浮窗，悬浮窗中包含多个按钮，点击不同按钮执行不同功能，点击悬浮窗外区域关闭软件。
按钮功能：将铃声、通知和闹钟的音量调到10%，以10为差值递增，至100%，共计十个按钮，点击按钮时，若当前模式为震动或静音，则将模式改为响铃。
UI：悬浮窗背景为圆角矩形，呈现毛玻璃效果，每个按钮为圆角正方形，颜色为渐变色，每排自左向右平滑渐变，一排五个，每个音量按钮的图标为一个喇叭加上数字，数字使用无衬线字体，整体表现出圆润、温和质感。
